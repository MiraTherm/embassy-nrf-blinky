# Basic Embedded Rust Dev-Container

Basic docker container for embedded Rust development.

Integrated:
- probe-rs v0.29.1
- Customized task buttons for VS Code
- Rust v1.90.0
- Blinky example for nRF52840 including
    - embassy-nrf
    - defmt
    - testing

Tested on WSL2 using `usbipd`.
